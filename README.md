# Enterprise Multi-Branch Network Infrastructure

## Overview

This project demonstrates a multi-branch office network architecture using MikroTik RouterOS and centralized Windows Server services.

## Environment

- 5 Branch Offices
- 150+ Users
- Site-to-Site VPN
- Active Directory
- Centralized File Services

## Technologies

- MikroTik RouterOS
- Windows Server
- Active Directory
- VPN
- VLAN
- DNS
- DHCP
